热更代理

只有继承这个代理类才能在热更层使用其泛型的变量值

继承此类使用数据层变量请用 self 代替 this

代理类中函数如果存在委托情况，请注意如仅存于非热更层的委托可能存在问题

此类事代理类，请不要使用代理类来直接声明变量或属性

当然如果真的有需要，可以使用 AgentDataPivot 类来存储变量实现热更层声明变量

另外虽然可以在热更层通过代理来直接传值达到既可以使用函数又可以读取值，但还是建议使用代理数据类来传递

热更层可以继承时间流接口，每次热更重载后时间流重置
